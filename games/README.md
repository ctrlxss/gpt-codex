# Breakout XP Game

This is a simple breakout-style game with experience points and randomized augments.

## Requirements
- Python 3.12+
- `pygame` library

Install dependencies with:

```bash
pip install pygame
```

## Running the game

```
python games/breakout.py
```

Use the mouse to move the paddle. Destroy blocks to gain XP. When you level up,
choose one of three random augments to improve your paddle, ball, or XP gain.
Clear all blocks to advance to the next level.

# Pong Deluxe

A browser-based version of Pong with simple graphics and sound. No additional dependencies are required; just open `games/pong/index.html` in any modern browser.

Controls:
- **A** or **Left Arrow** to move left
- **D** or **Right Arrow** to move right
- **P** to pause

High scores are saved to local storage.
