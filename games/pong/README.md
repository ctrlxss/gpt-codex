# Pong Deluxe

Simple browser-based Pong game with basic menu, pause functionality and high score persistence.

## How to Play

1. Open `index.html` in a web browser.
2. Click **Start Game**.
3. Move paddle left and right using **A / D** or **Left / Right arrows**.
4. Press **P** to pause the game.
5. Score as many points as possible before the CPU reaches 5.

High score is saved in your browser's local storage.
