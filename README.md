# Pong with Self-Training AI

This repository contains a simple Pong game written in Python using [pygame](https://www.pygame.org/). The right paddle uses a basic Q-learning algorithm and learns while you play.

## Requirements

- Python 3.12+
- `pygame` (install with `pip install pygame`)

## Running the Game

```bash
python3 pong_ai.py
```

Controls: use **W** and **S** to move your paddle up and down. The AI will train automatically as the game runs.
